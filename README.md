**运行环境：**

- 系统：Win10
- 开发环境：VS2017
- OpenGL版本：4.6.0 NVIDIA 441.66
- OGLU工具库版本：1.2.2.0 Microsoft Corporation
- CUDA版本：10.2
- 使用GPU device 0: GeForce GTX 950M
- SM的数量：5
- 每个线程块的共享内存大小：48 KB
- 每个线程块的最大线程数：1024
- 每个EM的最大线程数：2048
- 每个EM的最大线程束数：64

**运行方法：**

打开bin中的可执行文件，选择4种模式，能看到动画渲染效果，并且运行几分钟后能看到一万次碰撞检测平均时间。
